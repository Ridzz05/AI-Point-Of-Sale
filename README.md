# SMART_POS // NEURAL_LEDGER_SYSTEM

> **"VOID THE FLUFF. COMMIT THE DATA."**

SmartPOS is a high-intensity, AI-integrated Point of Sales system designed for maximum utility and zero visual distraction. Built on the principles of **Solid Nihilism** and **Neo-Brutalist** architecture, it transforms transaction management into a raw, powerful terminal experience.

---

## 💀 DESIGN_PHILOSOPHY: SOLID_NIHILISM
The interface is a rejection of modern corporate softness.
- **ZERO_RADIUS:** All corners are sharp. No rounded edges.
- **PURE_CONTRAST:** High-intensity Black & White palette with Toxic Yellow accents.
- **HEAVY_WEIGHT:** 4px-8px solid borders. Everything has weight.
- **BRUTAL_SHADOWS:** Solid drop shadows without blur. No gradients. No transparency.

---

## ⚡ CORE_FEATURES

### 1. AI_COMMAND_CENTER (VOICE + TEXT)
- **Neural_Parsing:** Input transactions using Natural Language (e.g., *"Jual 2 Americano dan 1 Donat"*).
- **Voice_Core:** Real-time microphone input with **stuttery-blink** visual feedback.
- **Multi-Intent:** Handles sales, inventory checks, and business analysis in one terminal.

### 2. REALTIME_FOCAL_DASHBOARD
- **Massive_Metrics:** Key financial data displayed with extreme hierarchy (text-9xl).
- **System_Monitor:** Real-time monitoring of memory usage, system latency, and transaction logs.
- **Inventory_Watchlist:** Automated stock alerts with "Toxic Yellow" highlight triggers.

### 3. BRUTAL_INVENTORY_LEDGER
- **Dense_Grids:** Every cell is bordered. Every piece of data is visible.
- **Mechanical_Interaction:** Buttons that "press" physically into the screen via solid shadow translation.

## 🚀 SYSTEM_DEPLOYMENT

### Prerequisites
- PHP 8.2+ & Composer
- Node.js 20+ & NPM
- OpenRouter API Key (Set in `.env`)

### Installation_Sequence
1. **Clone & Install Backend:**
   ```bash
   git clone <repository_url>
   composer install
   php artisan migrate --seed
   ```

2. **Initialize Frontend:**
   ```bash
   cd frontend
   npm install
   ```

3. **Execute Runtime:**
   - Terminal 1: `php artisan serve`
   - Terminal 2: `cd frontend && npm run dev`
